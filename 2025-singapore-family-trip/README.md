# 2025 Singapore Family Trip App

這是一個專為 2025 新加坡親子遊設計的行動版網頁應用程式 (Mobile Web App)。

## ✨ 主要功能
*   **總覽 (Overview)**：倒數計時、即時航班資訊 (星宇航空)、飯店資訊、智慧「下一步」行程提示。
*   **行程 (Itinerary)**：5 天詳細行程表、Google Maps 導航連結、行程編輯與排序功能。
*   **票券 (Tickets)**：電子票券收納 (支援圖片與 PDF)、分類管理 (機票/動物園等)、全螢幕檢視器。
*   **工具 (More)**：新加坡天氣預報 (API 串接)、資料備份/還原、A4 格式列印 (行程表/機票憑證)。

## 🚀 如何執行

1.  **安裝套件**
    ```bash
    npm install
    ```

2.  **啟動開發伺服器**
    ```bash
    npm run dev
    ```

3.  **建置正式版**
    ```bash
    npm run build
    ```

## 📱 手機安裝 (PWA)
在手機瀏覽器 (Safari/Chrome) 開啟後，選擇「加入主畫面 (Add to Home Screen)」，即可像原生 App 一樣全螢幕執行。
